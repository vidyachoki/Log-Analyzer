# 🔍 Log Analyzer — Production Support Tool

> An intelligent log analysis tool that automatically detects errors, classifies severity levels, and generates Root Cause Analysis for production support engineers.

## 🚀 Live Demo
👉 **[Try it Live](https://vidyachoki.github.io/ai-log-analyzer/)**

## ✨ Features
- Paste logs or upload .log/.txt files
- Auto-detects CRITICAL / ERROR / WARNING severity
- Generates Root Cause Analysis for each error
- Provides investigation steps to follow
- Shows error timeline
- Export report as text file
- 3 built-in sample log types to try instantly

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript (No frameworks needed)
- 100% browser-based — no backend, no API key required
- GitHub Pages deployment

## 📊 Sample Log Types Included
- Database errors (connection timeout, deadlock, OOM)
- AWS CloudWatch (Lambda timeout, S3 errors, ECS failures)
- API/Application (HTTP 500, 401, 429, Redis failures)
